This script adds the possibility to search for an EVN (vehicle number) on bahn.expert. 
You can enter an EVN or a part of it into a field  and let your browser search through the timetable of a station.I tested it with firefox and it works. The option to display the EVN has to be enabled in the settings.

How to install:

1. Add the tampermonkey extension to a browser that supports it.

2. Create a new userscript. Add the content of the file EVN-Finder.user.js.

3. Ensure that the settting for displaying EVN is enabled       <img width="1287" height="934" alt="setting" src="https://github.com/user-attachments/assets/6fb2850a-a8e8-4100-b06c-0af3115b3a63" />


How to use it:

1. Navigate to bahn.expert
3. Type in the number or parts of it  
4. Click on Scan starten
