This script adds the possibility to search for an EVM on bahn.expert. 
You can enter an EVN or a part of it into a field  and let your browser search through the timetable of a station.
I tested it with firefox and it works. The option to display the EVN has to be enabled in the settings.

How to install:

Add tampermonkey to a browser that supports it.

Create a new userscript. Add the content of the file EVNsearch-Bahn.expert-1.user.js 

That should be it. It´s ready to use then.

Just type in the number and start scanning
